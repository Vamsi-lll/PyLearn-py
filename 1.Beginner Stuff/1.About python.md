<h1>What is Python and Why you should Learn it ?</h1>

## What is Python?
Python is a popular programming language that is simple, powerful, and easy to read.  
It was created in 1991 by Guido van Rossum and is used everywhere today — from building websites and apps to analyzing data, creating games, and even powering artificial intelligence.

Python’s syntax looks a lot like plain English, which makes it beginner‑friendly and fun to learn.

## Why you need to learn Python?
- Easy to Learn: The code is clean and simple, perfect for beginners.
- Versatile: You can use Python for web development, data science, machine learning, automation, and more.
- Huge Community: Millions of developers use Python, so you’ll always find tutorials, libraries, and support.
- High Demand: Python developers are in demand across industries, making it a great career skill.
- Fun Projects: From games to apps, Python lets you build cool things quickly.
- Most importantly, you should learn Python because it lets you create your own projects and have fun while doing it.

Fun Fact
