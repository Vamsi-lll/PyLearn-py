# Installing Python on Your System 

To write and run Python code, you first need to install Python on your system.

## Step 1: Download Python
- Go to the official Python website:
  https://www.python.org/downloads/
- Download the **latest stable version** of Python for your operating system.

## Step 2: Install Python
- After downloading, **double-click the installer**.
- If possible, **run it as Administrator**.

## Step 3: Important Installation Settings
- In the installation window:
  - ✅ **Check the box** that says **“Add Python to PATH”**
    > This allows you to run Python from the command line.
  - Keep the **default installation path** (recommended for beginners).

- Click **Install Now** and wait for the installation to finish.

## Step 4: Verify Installation
- Open **Command Prompt / Terminal**
- Type:
  ```bash
  python --version
